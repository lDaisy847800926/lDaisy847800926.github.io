ls -l 以长格式展示文件的信息
ls -h 显示文件的大小

5、查看文件
cat 查看文件
head 查看文件内容的开头
tail 查看文件内容的尾部
less 按住enter走一下 敲空格走下一页
more 全屏显示
wc a.txt 一次显示文件的行数，字节数，文件名信息

echo $PATH 输出内容 查看环境变量

6、mkdir 创建目录
mkdir 创建目录 相对/绝对路径都可以
mkdir /a/b/c/d -p 递归创建，没有存在的也闯出来了
touch 创建文本

7、rmdir 删除空目录
rmdir -r 删除非空目录
rm 删除普通文件 空的
rm -r 删除目录 递归删除
rm -ri i是提醒你是否要删除

8、复制
cp ./a.txt test/ 将a.txt复制到test目录下
cp -r（递归）目录的复制

9、剪切
mv 剪切文件/剪切目录,移动文件或目录 mv 文件 目标目录
mv a.txt d.txt 将a 重命名为b

10、ln 建立链家文件 也就是Windows下的快捷方式
软链接 ln -s d.txt d1 d1是d的软连接，源文件删除，软链接失效，源软件链接和源文件不在统一目录，源文件要使用绝对路径
硬链接 ln d.txt d1 源文件删除，硬链接还可以用，硬链接和源文件的大小是一样的

11、grep 管道
ls grep | d 查找带d的
find 查找文件
find 文件路径 / -name 根据文字来查找
find ./ -name ‘*.sh’ 查找目录下所有的后缀为.sh的文件

12、解压缩
压缩分为两步：打包、压缩
tar 打包或拆包文件 tar -cvf (f是文件名)+打包名.tar
tar 解包文件 tar -xvf +解包名
压缩 gzip test.tar test.tar.gz
gzip -r test.tar taet.gz -r(压缩所有的子目录)

解压分为两步：解压、拆包
gzip -d test.tar.gz 解压文件
tar -zcvf test.tar.gz * 压缩和打包功能
tar -zxvf test.tar.gz 解压
zip -r test 压缩
unzip test.zip 解压
